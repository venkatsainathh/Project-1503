# Our Love Story - 3 Years Adventure Game

A personalized interactive birthday game celebrating 3 years of love and adventures together. Built with HTML, CSS, and JavaScript. No build step required.

## How to Play

1. **Open the game**: Double-click `index.html` to open in your browser, or run a local server:
   ```bash
   python3 -m http.server 8080
   ```
   Then visit: http://localhost:8080

2. **Begin the journey**: Click "Begin Our Journey" on the landing screen.

3. **Read the story**: Each chapter tells a part of your love story. Read it and click "Start Challenge".

4. **Complete challenges**: Each level has a mini-game to complete before moving forward.

5. **Reach the end**: Complete all 10 chapters to unlock the birthday surprise!

## Chapters

1. **COVID Connection** - Memory matching
2. **College & Birthday Cake** - Light the candles
3. **A New Beginning** - Type your anniversary date
4. **College Days Together** - Diwali sequence
5. **Dubai & Long Distance** - Bridge the distance
6. **UKUSA Reunion** - Memory matching
7. **USA & Reconnection** - Collect the hearts
8. **Urban Company & The Surprise** - Pack the surprise
9. **Unbreakable Bond** - Support sequence
10. **Our Forever** - Open your heart

Enjoy the journey! 💕

## Publish on GitHub

1. Create a new repository on [GitHub](https://github.com/new) (e.g. `birthday-adventure-game`). Do **not** initialize with a README if this project already has one.

2. From your project folder, run:
   ```bash
   git add .
   git commit -m "Initial commit: birthday adventure game"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```
   If the remote is already set up, you only need:
   ```bash
   git add .
   git commit -m "Update: full birthday game"
   git push -u origin main
   ```

3. To host the game for free: go to your repo → **Settings** → **Pages** → set source to **Deploy from a branch** → choose branch `main` and folder `/ (root)` → Save. Your game will be at `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`.
